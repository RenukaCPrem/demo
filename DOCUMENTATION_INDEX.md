## 🔍 How to Search Within Documents

If you're in a document and need to find something:

**Windows/Linux:**
- Press `Ctrl+F`
- Type what you're looking for
- Press Enter

**Mac:**
- Press `Cmd+F`
- Type what you're looking for
- Press Enter

For example:
- Search "Firebase" to find all Firebase mentions
- Search "error" to find troubleshooting sections
- Search "CSS" to find styling information
