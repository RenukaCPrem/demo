## Step 4: Test Your App

After following the checklist:

---

## Step 5: If Something Breaks (WHEN it will)
**Time: 15-30 minutes**

When you run into problems (and you will - that's normal):

1. **Open the troubleshooting guide:** [TROUBLESHOOTING_GUIDE.md](TROUBLESHOOTING_GUIDE.md)
2. **Find your error** in the "Common Problems" section
3. **Follow the solution**
4. **Test again**

Most common issues:
- "Firebase SDK not loaded" → Missing `<script>` tags
- "User not found" → Wrong email/password
- "Permission denied" → Firestore rules not published
- "Deployment fails" → Not logged into Firebase

## Common Beginner Mistakes

❌ **Don't:**
- Try to memorize all the code (you don't need to!)
- Skip reading the guides (they're written for you!)
- Copy code without understanding what it does
- Deploy without testing locally first
- Give up on the first error (debugging is normal!)

✅ **Do:**
- Copy code from CODE_EXAMPLES.md
- Read error messages carefully
- Test each step before moving to the next
- Ask for help when stuck
- Take breaks if frustrated

## Getting Help

**Before asking your teacher:**
1. Check TROUBLESHOOTING_GUIDE.md
2. Look at browser console (F12)
3. Try the solution yourself
4. Write down what you tried

**When asking for help:**
- Show them the exact error message
- Tell them what you already tried
- Point to the specific code that's broken
- Your teacher will help much faster!
